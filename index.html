The Indices

The un-copyable layer. Some numbers are computed from the verified corpus; others from human sources no competitor can scrape. Here is exactly how each one earns its trust status.

How a number earns its status
Verified
Reserved for corpus-derived facts. Every input traces to a human-reviewed CONFIRM citation. Only the Regulatory Pressure Index qualifies today — because a legal obligation has a single source of truth. No number reaches Verified without human sign-off. That rule is permanent.
Corroborated
The ceiling a street index can reach. Published methodology, ≥3 independent sources agreeing within tolerance, timestamped and reproducible. A measurement has no single legal truth — so it is corroborated, never "verified." Honest, and defensible in an investment committee.
Indicative
Early or thin-sourced. A single desk, a first read, a widening you should watch but not yet bank. Flagged so no one mistakes a signal for a settled number.
Promotion path Indicative → Corroborated: widen the source panel, publish the method, set a divergence tolerance, archive every reading, human sign-off before publish. It is the CONFIRM promotion gate, applied to data.
